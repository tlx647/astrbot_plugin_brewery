# 酿酒大师 AstrBot 插件
> 沉浸式酿酒经营玩法插件，为你的机器人添加完整的酿酒、陈酿、交易、排行互动系统

## ✨ 功能特色
- 📦 **完整经济体系**：初始金币、每日签到、材料购买、酒品出售，闭环式经济玩法
- 🍶 **丰富配方系统**：内置 8 种国酒+洋酒配方，支持自定义材料、时长、售价、品质
- 🏺 **陈酿增值玩法**：酒品放入酒窖随时间自动提升品质，星级越高售价越高，周期可配置
- 🏆 **双排行榜系统**：金币富豪榜、酿酒数量大师榜，带动社群互动氛围
- ⚙️ **高度可自定义**：所有参数支持后台可视化配置，无需修改代码
- 💾 **数据持久化存储**：用户数据自动本地保存，支持异常恢复，原子写入防止损坏
- 🌐 **全中文友好体验**：材料、指令、提示全部中文化，支持中文材料名指令输入
- 🛡️ **健壮的异常处理**：边界场景全覆盖，参数错误友好提示，后台协程异常不崩溃

## 🚀 安装方法
### 方法一：后台上传安装（推荐）
1. 下载插件压缩包 `astrbot_plugin_brewery.zip`
2. 进入 AstrBot 管理后台 → 插件管理 → 上传插件
3. 选择下载好的压缩包，等待安装完成
4. 在插件列表中找到「酿酒大师」，点击启用即可

### 方法二：手动解压安装
1. 解压插件文件到 AstrBot 数据目录下的 `plugins/astrbot_plugin_brewery` 文件夹
2. 重启 AstrBot 或在后台重载插件
3. 插件自动加载并初始化数据

## ⚙️ 配置说明
所有配置项均可在 AstrBot 后台可视化编辑，无需修改代码。

### 经济系统配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 新用户初始金币 | 1000 | 首次使用插件的用户默认金币数量 |
| 每日签到金币 | 50 | 用户每日签到可获得的金币奖励 |

### 材料市场配置
| 材料 | 默认单价 | 说明 |
|------|----------|------|
| 高粱 | 10 | 酱香型白酒核心原料 |
| 大米 | 8 | 黄酒、米香型白酒原料 |
| 小麦 | 12 | 多粮浓香、清香型原料 |
| 玉米 | 9 | 伏特加等烈性酒原料 |
| 葡萄 | 15 | 红葡萄酒、白兰地原料 |
| 大麦 | 11 | 威士忌、麦芽酒原料 |
| 酒曲 | 5 | 酿造必需发酵材料 |
| 泉水 | 3 | 所有酒品基础材料 |

### 配方配置
支持自定义修改每个配方的以下参数：
- 酒品名称
- 各种材料所需数量
- 酿造时长（秒）
- 基础售价（金币）
- 基础品质星级（1-5）
- 每陈酿周期品质加成

### 陈酿系统配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 陈酿周期 | 300 秒 | 每经过多久时间提升一次酒品品质 |
| 最高品质星级 | 5 | 酒品可达到的品质上限 |
| 默认酒窖槽位 | 3 | 每个用户默认拥有的酒窖格子数量 |

## 📁 文件结构

- 用户数据：`data/plugin_data/astrbot_plugin_brewery/users.json`
- 配置文件：`data/config/astrbot_plugin_brewery_config.json`

## 🚨 注意事项
1. 插件依赖 AstrBot v4.0+ 版本运行
2. 请勿直接手动编辑用户数据文件，可能导致数据损坏
3. 修改配方配置后，已酿造完成的酒品属性不受影响
4. 陈酿后台协程每分钟执行一次，品质提升最多有 1 分钟延迟
5. 签到按北京时间自然日刷新，跨天无需重启插件
